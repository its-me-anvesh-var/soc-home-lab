# 🔧 Wazuh Manager Installation Guide

**OS:** Ubuntu 22.04 LTS  
**Wazuh Version:** 4.7  
**Author:** Anvesh Raju Vishwaraju

---

## Prerequisites

- Ubuntu 22.04 LTS (fresh install)
- Minimum 4GB RAM, 50GB disk
- Internet access during install
- Static IP assigned (e.g. 192.168.56.10)

---

## Step 1 — System Preparation

```bash
# Update system
sudo apt update && sudo apt upgrade -y

# Set hostname
sudo hostnamectl set-hostname wazuh-manager

# Disable swap (recommended)
sudo swapoff -a
sudo sed -i '/ swap / s/^/#/' /etc/fstab
```

---

## Step 2 — Install Wazuh (All-in-One)

```bash
# Download installer
curl -sO https://packages.wazuh.com/4.7/wazuh-install.sh

# Run all-in-one installation
# Installs: Wazuh Manager + Indexer + Dashboard
sudo bash wazuh-install.sh -a

# Save the admin credentials shown at the end!
# Username: admin
# Password: <auto-generated>
```

---

## Step 3 — Verify Installation

```bash
# Check all services
sudo systemctl status wazuh-manager
sudo systemctl status wazuh-indexer
sudo systemctl status wazuh-dashboard

# Check manager is listening
sudo netstat -tulnp | grep wazuh

# Expected ports:
# 1514/udp  - Agent communication
# 1515/tcp  - Agent enrollment
# 55000/tcp - REST API
# 443/tcp   - Dashboard
```

## Step 5 — Configure Email Alerts (Optional)

```bash
sudo nano /var/ossec/etc/ossec.conf
```

Add inside `<global>` section:
```xml
<email_notification>yes</email_notification>
<smtp_server>smtp.gmail.com</smtp_server>
<email_from>wazuh@yourdomain.com</email_from>
<email_to>security-team@yourdomain.com</email_to>
<email_maxperhour>10</email_maxperhour>
```

```bash
# Restart manager
sudo systemctl restart wazuh-manager
```

---

## Step 6 — Add Custom Rules

```bash
# Copy custom rules
sudo cp detection-rules/wazuh/custom-rules.xml /var/ossec/etc/rules/
sudo cp detection-rules/wazuh/decoder.xml /var/ossec/etc/decoders/

# Test rules syntax
sudo /var/ossec/bin/ossec-logtest

# Restart
sudo systemctl restart wazuh-manager
```

---

## Step 7 — Enroll Windows Agent

On the Windows target machine:
```powershell
# Download agent MSI from:
# https://packages.wazuh.com/4.x/windows/wazuh-agent-4.7.0-1.msi

# Install via PowerShell (replace IP with your manager IP)
msiexec.exe /i wazuh-agent-4.7.0-1.msi /q `
  WAZUH_MANAGER="192.168.56.10" `
  WAZUH_AGENT_NAME="Windows-Victim"

# Start agent service
NET START WazuhSvc
```

---

## Step 8 — Enroll Linux Agent

On the Linux target:
```bash
# Add Wazuh repo
curl -s https://packages.wazuh.com/key/GPG-KEY-WAZUH | gpg --dearmor -o /usr/share/keyrings/wazuh.gpg
echo "deb [signed-by=/usr/share/keyrings/wazuh.gpg] https://packages.wazuh.com/4.x/apt/ stable main" | sudo tee /etc/apt/sources.list.d/wazuh.list

# Install agent
sudo apt update
sudo WAZUH_MANAGER="192.168.56.10" WAZUH_AGENT_NAME="Ubuntu-WebServer" apt install wazuh-agent -y

# Enable and start
sudo systemctl daemon-reload
sudo systemctl enable wazuh-agent
sudo systemctl start wazuh-agent
```
